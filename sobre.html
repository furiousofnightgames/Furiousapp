CONHEÇA O FURIOUS APP

Tecnologia avançada de gerenciamento de downloads desenvolvida para máxima performance.

O QUE É?

O Furious App: Game Hub Edition é um ecossistema local-first de gerenciamento de downloads desenvolvido com a stack FastAPI + Vue 3 + Electron. Inspirado na agilidade de Downloaders de ponta e na potência do aria2c, o sistema centraliza múltiplas fontes JSON em uma biblioteca unificada com interface "Netflix-like".

DIFERENCIAIS TÉCNICOS DE ELITE

BUFFERED PERSISTENCE

Arquitetura de escrita em lote que reduz a carga no disco em 100x, garantindo uma UI fluida a 60FPS mesmo durante scans massivos de metadados.

SONDAGEM UDP LIVE

Motor de análise sonora que contata trackers via UDP em tempo real para obter a saúde exata de Magnets e Torrents antes de reservar banda.

INTEGRIDADE CARD-SYNC

Lógica 'Quick-Check' que audita o disco físico para detecção de arquivos fantasmas ou corrompidos, sincronizando automaticamente o tamanho real com a biblioteca.

PORTABILIDADE TOTAL

O instalador traz binários portáveis integrados do Python 3.10.5+, Node.js 18+ e aria2c 1.37.0+. Zero dependências no Windows.

FAQ TÉCNICO & SEGURANÇA
O SISTEMA É SEGURO E PRIVADO?

Sim. O Furious App é 100% Local-First. Seu histórico de downloads, favoritos e fontes JSON permanecem exclusivamente no seu computador em um banco de dados SQLite encriptado localmente.

COMO FUNCIONA A LIMPEZA PÓS-INSTALAÇÃO?

Após concluir a instalação de um Item via Smart Installer, o App oferece a remoção segura dos arquivos de instalação (Itens obsoletos) para liberar espaço, mantendo o registro e atalhos na sua biblioteca.

A REGRA X1.25 DE ESPAÇO

Para evitar falhas de instalação, nosso assistente calcula se você possui espaço suficiente para: [Arquivo Baixado] + [Jogo Instalado], prevenindo que você gaste banda em algo que não caberá no disco.
